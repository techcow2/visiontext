## Installation & Setup

1. **Install**: Search "VisionText" in VS Code Extensions Marketplace
2. **Restart**: Reload VS Code to activate the extension
3. **Use**: Right-click any image or use the command palette (Ctrl+Shift+P)
4. **Configure**: Adjust settings in Preferences > Extensions > VisionText

**No API key required** - VisionText works out of the box with generous free tier limits!
